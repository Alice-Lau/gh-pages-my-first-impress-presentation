impress.js

What's impress.js?

A Javascript framework , which utilize the power of CSS transforms and transitions to create customized, unique looking presentation slides.

Inspiration of the idea

The idea was inspired by the presentation software Prezi

Prezi

  • require Flash
  • fee required too access all features

Impress.js

  • CSS transforms, transitions
  • completely FREE

Cross-Browser Support

  • Internet Explorer 10+
  • Firefox 10+
  • Chrome
  • Safari
  • only know exception: Opera Mini

Solution

Impress is supported across most updated versions of popular browsers.

One known exception is opera mini.

However, users on unsupported browsers can still view an unanimated version of the presentation

if we include div with class of "impress-not-support" and "fallback-message" at the beginning of the HTML.

How to make an Impress.js Presentation?

  1. download the impress.js script file

  2. link the external script to our html file

  3. initiate the script

Let move onto the main construction of Impress.

First, we will have to download Impress JavaScript file and save it in our presentation folder.

In our HTML we also have to include external link to the js file, as well as initiate the Javascript using the init function.

The Main HTML Structure

  1. create a div with an id "impress"

  2. create internal div with the class "step" inside "impress" to indicate individual slide

Now, onto the visible part of the presentation.

We would create a div with an id of impress, within which we places each of our individual slides. The slide should all have the class named step.

Other than these 2 requirement, we basically style our slide the same as normal webpages.

Next slide is the fun part, where we finally come to the part in customizing the transitions of our slides.

Customizing the Transition

As shown in the code demonstration, Impress.js document recommended us to place our specification inside the html element tag

There are a total of 7, argueably 8 attributes that we can use in any combination we want.

For 2D Effect, we have data-x, which moves the slide left/right; data-y, which moves the slide up/down; and data-rotation, which turn the slide clockwise/counter-clockwise along a 2D plane

I am going to skip over the 3D effects for now, as I will explain them in more detail in following slides

For other effects, data-z represent the perspective value, which move the slide forward/closer to the user, the larger the value, and backward/further away from the user, the smaller the value

Data-Rotation-X

data-rotation-x spins a slide along a horizontal axis (stetching from east to west).

see demo

Data-Rotation-Y

data-rotation-y spins a slide along a vertical axis.

see demo

Data-Rotation-Z

data-rotation-z spins a slide along a horizontal axis (stretching from north to south).

see demo

Styling

  1. class="past"

  2. class="future"

  3. class="active"

Most CSS styling are the same as how we would normally write them for typical webpages

However, there're 3 special classes created by impress.js

The "past" and "future" class represents slides before or after the current slide. Usually we will give it an opacity less than 1 to make the slides look translucent. While the active slide will recieve an opacity of 1

These classes are changed dynamically by impress.js script.
